# Getting Started with Panel Flow

Panel Flow is a professional comic book script writing tool designed for sequential art writers.

## Quick Start

1. **Sign in** with your Google account
2. **Create a series** from the dashboard
3. **Create an issue** within your series
4. **Start writing** using the three-column editor

## The Editor

The issue editor has three main sections:

### Navigation (Left Panel)
- View your issue structure: Acts → Scenes → Pages
- Click any page to edit it
- Drag to reorder scenes and pages
- Add new acts, scenes, and pages

### Editor (Center Panel)
- Write panel descriptions and dialogue
- Add characters, captions, and sound effects
- Each panel is automatically numbered
- Changes auto-save as you type

### Toolkit (Right Panel)
- **Context Tab**: View and edit issue theme, motifs, and stakes
- **AI Chat Tab**: Get writing assistance from the AI
- **Alerts Tab**: Check for continuity issues

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘/Ctrl + S | Save confirmation |
| ⌘/Ctrl + Z | Undo |
| ⌘/Ctrl + Shift + Z | Redo |
| ⌘/Ctrl + F | Find & Replace |
| Tab | Next field |
| Shift + Tab | Previous field |
| Escape | Close modal / collapse panel |

## Exporting

Export your scripts in multiple formats:
- **PDF**: For printing and sharing
- **DOCX**: For editing in Word
- **TXT**: Plain text format

Click the export buttons in the header to download.

## Characters & Locations

Before writing, set up your:
- **Characters**: With names, descriptions, and speech patterns
- **Locations**: With descriptions and visual details
- **Plotlines**: Color-coded threads to track through your story

These will appear in dropdowns throughout the editor.

## Tips

1. **Auto-save**: Changes save automatically every 2 seconds
2. **Version History**: Access past versions via the History link
3. **Import**: Paste existing scripts via the Import feature
4. **Weave View**: Visualize your plotlines across pages

## Mobile Usage

On mobile devices, use the Navigation/Editor/Toolkit tabs at the top to switch between views.

## Need Help?

- Use the AI chat in the Toolkit for writing assistance
- Check the Continuity tab for potential issues
- View the Outline page for a bird's-eye view of your series
